# SnaptureBD — Luxury Photography Studio

A polished, production-ready photography studio website built with **React + Vite**. Fully responsive, animated, and designed with a refined black-and-gold editorial aesthetic.

## Features

- **Home** — cinematic hero, studio intro, services, featured gallery, stats, testimonials, CTA
- **About** — philosophy, team, awards, stats
- **Portfolio** — filterable gallery with lightbox
- **Services** — specialties + 4-step creative process
- **Pricing** — 3-tier collections + FAQ
- **Journal (Blog)** — featured post + grid + newsletter signup
- **Contact** — full enquiry form with validation
- Sticky glass navbar with mobile menu, scroll-reveal animations, Ken Burns hero, smooth transitions

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18
- React Router v6
- Vite 5
- Pure CSS (no framework) with custom design tokens

## Customizing

- Colors/fonts: `src/styles/global.css`
- Content/copy: `src/data/content.js`
- Images: currently using Unsplash URLs — replace with your own assets in `public/`
